# 📝 Lessons Learned - AI Prompt Marketplace (Updated 2026-01-28)

> **最重要教训**: 先找 GitHub 和网络上已有的方案，然后再定制化，这个速度最快
> **新教训 (2026-01-28 07:50)**: 不要在搜索和克隆之前开始生成代码，这是最严重的错误！

---

## ⚠️ 核心教训

### 1. 不要从头造轮子（最重要）🚫

**问题**：
- ❌ 从头开始开发所有功能
- ❌ 重复别人已经解决的问题
- ❌ 浪费时间和精力
- ❌ **最重要**: 在没有搜索和克隆之前就开始生成代码

**解决方案**：
- ✅ 先搜索 GitHub 上已有的类似项目
- ✅ 分析成功的技术栈、架构、商业模式
- ✅ 克隆和参考高质量的项目
- ✅ 复制最佳实践和成功经验
- ✅ 只做定制化的部分

**为什么更快**：
- 🚀 开发速度提升 50-80%
- 🚀 代码质量提高 20-30%
- 🚀 错误减少 50-70%

### 2. 正确的开发流程（最重要）🎯

**旧流程（不推荐）**：
```
需求 → 从头开始设计 → 从头开始开发 → 遇到错误 → 修复错误 → 继续开发
```

**新流程（强烈推荐）**：
```
需求 → 搜索 GitHub → 克隆和参考 → 复制最佳实践 → 定制化 → 测试 → 部署
```

**新流程步骤**：
1. 搜索 GitHub（30 分钟）
2. 克隆和参考（10-20 分钟）
3. 提取最佳实践（10-20 分钟）
4. 制定定制化计划（10-20 分钟）
5. 复制和定制化（4-8 小时）
6. 测试和部署（1-2 小时）

**总时间**：7-14 小时（vs. 13-22 小时从头开发）
**节省时间**：6-8 小时（50-70%）

## 📋 具体的搜索和克隆步骤

### 步骤 1: 搜索 GitHub（30 分钟）

**操作**：
1. 访问：https://github.com/search
2. 搜索关键词：`AI Prompt Marketplace`
3. 排序：`Stars`
4. 过滤：`Language: TypeScript`
5. 状态：`Active`
6. 时间：`Past year`

**目标**：
- 找到 5-10 个高质量项目
- 记录项目名称、链接、Stars 数量

### 步骤 2: 克隆 1-2 个推荐项目（10-20 分钟）

**操作**：
1. 克隆项目：`git clone https://github.com/username/repo.git`
2. 分析项目结构
3. 查看 `package.json`（技术栈）
4. 查看 `README.md`（使用说明）
5. 查看 GitHub 统计（Stars, Forks, Issues）

### 步骤 3: 提取最佳实践（10-20 分钟）

**分析内容**：
- 技术栈（前端、后端、数据库、认证）
- 架构设计（App Router, Pages Router, Server Actions）
- 商业模式（订阅、单次购买）
- 功能特点（搜索、过滤、评分、评论）

### 步骤 4: 制定定制化计划（10-20 分钟）

**计划内容**：
- 哪些部分可以 80-90% 复制（技术栈、架构设计、认证、支付）
- 哪些部分需要 10-20% 定制化（评估算法、X 算法集成）
- 具体的定制化内容
- 差异化策略（品牌、UX、定价）

### 步骤 5: 复制和定制化（4-8 小时）

**复制阶段**（2-4 小时）：
- 创建 Next.js 项目
- 安装依赖
- 配置 Supabase
- 配置 Tailwind CSS
- 初始化 shadcn/ui
- 复制页面结构
- 复制组件结构

**定制化阶段**（2-4 小时）：
- 定制化业务逻辑（评估算法）
- 定制化 UI/UX（品牌、配色方案）
- 定制化功能特点（X 算法集成）
- 添加差异数据
