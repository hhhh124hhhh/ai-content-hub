## 📋 核心功能

### 1. 提示词展示 📺

**页面结构**：
- 🏠 首页 - 热门提示词展示
- 📋 分类列表 - 按类型浏览
- 🔍 搜索页面 - 全文搜索和过滤
- 📝 详情页面 - 完整提示词内容
- 💾 我的提示词 - 收藏和购买历史

**展示方式**：
- 卡片式展示（标题、描述、评分）
- 标签和分类
- 作者信息
- 点赞、转发、评论数

### 2. 评分系统 📊

**评分维度**（100 分制）：
- 实用性 (30%)
- 创新性 (25%)
- 完整性 (20%)
- 热度 (25%)

**等级划分**：
- A+ (90-100): 顶级
- A (85-89): 优秀
- B (75-84): 良好
- C (60-74): 一般
- D (0-59): 较差

### 3. 支付系统 💳️

**支付方式**：
- 💳️ Stripe（信用卡、借记卡）
- 💰 PayPal
- 💲 加密货币（USDT, USDC）

**支付流程**：
1. 用户选择套餐或提示词
2. 点击"购买"或"订阅"
3. 跳转到 Stripe Checkout
4. 支付成功后
5. 自动激活订阅或发送下载链接

### 4. 订阅管理 👤

**用户功能**：
- 查看当前订阅状态
- 升级/降级套餐
- 取消订阅（立即或到期时）
- 查看账单和支付历史

**管理功能**：
- 查看所有订阅用户
- 手动处理退款
- 创建优惠券和促销
- 发送通知邮件

### 5. 内容抓取 🤖

**抓取来源**：
- X (Twitter) - AI 提示词
- Reddit - r/ChatGPT, r/artificial
- Discord - AI 相关服务器
- 博客 - AI 研究者博客

**抓取频率**：
- 每 6 小时抓取一次
- 实时监控热门话题
- 按质量筛选和评分

### 6. 用户系统 👥

**注册/登录**：
- Email 注册
- Google 账号
- GitHub 账号

**用户资料**：
- 姓名、头像
- 收藏的提示词
- 购买历史
- 使用统计

## 💰 盈亏平衡点

### 计算公式

**盈亏平衡点** = 固定成本 / (单价 - 可变成本)

- **固定成本**: $13,128/年
- **可变成本**: $0（数字产品）
- **平均用户收入**: $49.9/月（基础版）
- **转化率**: 5%

**需要用户数** = 固定成本 / (月度收入 × 12 × 转化率)

**计算**:
- 固定成本: $13,128
- 月度收入: $49.9
- 年度收入: $598.8
- 转化率: 5%

**需要用户数** = $13,128 / ($598.8 × 0.05) = 438 用户

### 盈亏平衡

- **第 1 年**: 需要 438 付费用户
- **第 2 年**: 保守估计达到 100+ 用户，乐观估计达到 300+ 用户
- **第 3 年**: 继续增长，预计达到 500+ 用户

## 🚀 部署步骤

### 1. 前端部署（Vercel）

```bash
# 1. 安装 Vercel CLI
npm i -g vercel

# 2. 登录 Vercel
vercel login

# 3. 部署到 Vercel
cd frontend
vercel --prod

# 4. 配置域名（可选）
vercel domains add yourdomain.com
```

### 2. 后端部署（Render）

```bash
# 1. 连接 GitHub 仓库到 Render
# 访问：https://render.com/dashboard
# 点击 "New +" → "Web Service"
# 选择你的 GitHub 仓库

# 2. 设置环境变量
# MONGODB_URI=mongodb+srv://...
# REDIS_URL=redis://...
# STRIPE_SECRET_KEY=sk_live_...
# STRIPE_WEBHOOK_SECRET=whsec_...
# JWT_SECRET=your-jwt-secret

# 3. 部署
# Render 会自动检测 package.json 并部署
```

### 3. 配置 Stripe

```bash
# 1. 登录 Stripe Dashboard
# https://dashboard.stripe.com

# 2. 创建产品
# 创建以下产品：
# - 基础版订阅: $4.99/月
# - 标准版订阅: $9.99/月
# - 专业版订阅: $19.99/月
# - 企业版订阅: $99.99/月

# 3. 创建价格
# 为每个产品创建月度价格和年度价格

# 4. 配置 Webhook
# 将 Webhook URL 设置为：
# https://your-backend-url.com/api/stripe/webhook

# 5. 复制 API Keys
# Secret Key: sk_live_...
# Publishable Key: pk_live_...
```

### 4. 配置 Cloudflare（CDN + DNS）

```bash
# 1. 添加域名到 Cloudflare
# 访问：https://dash.cloudflare.com

# 2. 配置 DNS
# 添加 CNAME 记录：
# api: your-backend-url.com
# www: your-frontend-url.com

# 3. 启用 Cloudflare CDN
# 启用 Automatic HTTPS
# 启用 Caching
```
